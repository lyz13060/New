网页内容嵌入方法

学习如何在网页中显示其他网页的内容，包括iframe、object、embed和AJAX等方法。

1. iframe 标签

iframe（内联框架）是最常用的嵌入方法，可以在当前页面中创建一个包含另一个文档的内联框架。

<iframe src="https://example.com" width="100%" height="400"></iframe>

优点：简单易用，广泛支持，可设置尺寸和边框。

缺点：可能被网站通过X-Frame-Options阻止加载。

2. object 标签

object标签用于嵌入各种对象，包括其他网页、图像、音频、视频等。

<object data="https://example.com" width="100%" height="400">
  <p>您的浏览器不支持object标签</p>
</object>

优点：更语义化，可提供备用内容。

缺点：支持不如iframe广泛。

3. embed 标签

embed标签用于嵌入外部内容，如插件、Flash或PDF文件。

<embed src="https://example.com" width="100%" height="400">

优点：简单，无需闭合标签。

缺点：主要用于多媒体内容，对网页支持有限。

4. AJAX 加载

使用JavaScript通过AJAX获取其他网页的内容，然后插入到当前页面中。

fetch('https://example.com')
  .then(response => response.text())
  .then(html => {
    document.getElementById('content').innerHTML = html;
  });

优点：更灵活，可控制加载内容和样式。

缺点：受同源策略限制，需要处理跨域问题。
